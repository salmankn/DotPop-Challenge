# 🎮 DotPop Challenge

**Connect Dots, Solve Puzzles!** Official website for the DotPop Challenge mobile puzzle game built with Flutter.

## 📋 Overview

DotPop Challenge is a fun and addictive mobile puzzle game where players connect matching dots to complete challenges, earn achievements, and unlock power-ups. This repository contains the official landing page with Privacy Policy and Terms of Service.

## ✨ Features

- **Landing Page** - Modern dark-themed game showcase with animated background
- **Privacy Policy** - Dedicated privacy policy page for Google Play Store
- **Terms of Service** - Dedicated terms of service page for Google Play Store
- **Smooth Animations** - Floating dot animations and gradient effects
- **Responsive Design** - Works on desktop, tablet, and mobile
- **Navigation** - Easy-to-use navigation bar with quick links to all pages

## 🚀 Deploy to GitHub Pages

```bash
# 1. Create a new public repository on GitHub
# 2. Upload all files to the repo
# 3. Go to Settings → Pages
# 4. Source: Deploy from a branch
# 5. Branch: main, folder: / (root)
# 6. Your site will be live at:
#    https://<username>.github.io/<repo-name>/
```
